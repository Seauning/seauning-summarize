# Dialog 工具

```jsx
function Demo() {
    const [visible, setVisible] = useState(false);
    return (
        <>
            <Dialog visible={visible} onClose={() => setVisible(false)} />
            <Button
                onClick={() => {
                    setVisible(true);
                }}
            >
                打开弹窗
            </Button>
        </>
    );
}
```
